BlackHole音频录制

Posted on 2021-05-29 Edited on 2021-07-14 Views: Disqus:

install

  • brew install blackhole-16ch
  • midi音频设置，添加多输出设备
  • 勾选 blackhole 16ch
  • 打开 quicktime，新建录制音频
  • 录制⏺️按钮边上，选择 blackhole 16ch：blackhole 16ch
  • 然后开始录制

问题

What is difference between 2ch and 16ch?

as far as i know there is not really a difference except the 16 channels supports 16 channels, so you can use 16ch hardware audio interfaces and route every input trough blackhole separate.
