Android Studio 走坑

Posted on 2024-01-12 Edited on 2025-11-01 Views:

flutter 开发，Android Studio 用的也比较多，走了不少坑，记录下来。

Android Studio 走坑

断点没反应

这就是最大的坑了，只要有奇怪的无法定位断点，别怀疑程序，别怀疑自己，就是 Android Studio 的问题。

  1. 断点多，会导致。今天就是断点多导致的，把断点都删了，留下要用的几个才行。用 Consumer 或者 bloc 都不能到达断点，耽误了整整一个多小时。
  2. 直接 hot load 的话，也会导致。必须重新编译运行才好

总是自动断开模拟器

自带的模拟器，性能太差，用的第三方或者手机，总是自动断开。

  1. 如果是手机的话，安装一个 wi-fi 插件，然后用 wifi 调试
  2. 如果是模拟器的话，之前感觉没那么频繁，但是更新 android studio 后，就频繁了，只能降低版本

安卓编译

暂时先加这里，后面再细化。

Error launching application on Android SDK built for x86.

build.gradle 的 ndk 配置，需要加上 x86

android {
    defaultConfig {
        ndk {
            abiFilters 'armeabi-v7a','arm64-v8a','x86_64','x86'
        }
