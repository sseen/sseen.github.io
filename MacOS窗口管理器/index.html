MacOS 窗口管理器

Posted on 2024-04-24 Edited on 2025-12-19 Views:

本来想写个窗口管理器的，但是发现已经有了，而且还是免费的，所以就不用自己写了。
就像那个日历一样，开源了之后，很多模仿者出现。
前两天刚下一个终身免费的 Align，加上之前买的 Magnet，不知道会不会用得上，但是还是收录了。

这里记录两个开源的。

试用了一下 Loop，感觉还是不习惯键盘+鼠标的操作。推荐一下自用多年的开源 Mac 窗口管理工具 Amethyst，GitHub 14k Stars: https://ianyh.com/amethyst/
特点是：纯键盘操作，自动规整 window 布局，多屏幕控制流畅。
ps. 也许看着快捷键很多，但是我日常最常用的就用4个左右
1. 切换布局是全屏还是 Column 排布
2. 切换当前的 Focus 屏幕
3. 切换当前的 Focus 窗口
4. 把窗口在屏幕间丢来丢去
躺在床上刷几天没读的 RSS，感觉信息爆炸了。
但看到这个工具之后，又猛地从床上坐起来打开电脑试试，还真好用。
Loop - 一款开源的 Mac 窗口管理软件，快速调整窗口大小、窗口位置，无需拖动贴边，交互感觉很像优化过后的 Win 11 的窗口管理，动画流畅丝滑。
仓库地址：https://github.com/MrKai77/Loop
