Xcode Build Variables

Posted on 2021-11-09 Edited on 2025-12-19 Views:

有时候需要设置 xcode 几个 shell 脚本，会用到各种内置的 build 参数。

比如 crashlytics 上传 dsym 时候，简单的判断需要用到；

# 本地调试不上传
# demo App 用 demo 配置
# App 用 正式配置
bundleID=${PRODUCT_BUNDLE_IDENTIFIER}
configuration=${CONFIGURATION}
id="com.xxx.demo"
fireBaseFileName="GoogleService-Info.plist"
# 空代表是本地调试
# 不然本地调试很慢必须有科学网络才能调试
echo "ssn shell"
echo $configuration
if [ "$configuration" = "xxx-Demo" ]; then
    echo "demo"
    fireBaseFileName="GoogleService-Info-demo.plist"
fi
# if [ -z "$bundleID"]; then
if [ "$configuration" = "Debug" ]; then
    echo "kong"
    fireBaseFileName="GoogleService-Info-demo.plist"
else 
    echo "upload"
    "${PODS_ROOT}/FirebaseCrashlytics/upload-symbols" -gsp "${PROJECT_DIR}/EndZone/${fireBaseFileName}" -p ios "${DWARF_DSYM_FOLDER_PATH}/${DWARF_DSYM_FILE_NAME}"
fi
