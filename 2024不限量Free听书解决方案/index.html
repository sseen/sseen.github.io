2024 不限量 Free 听书解决方案

Posted on 2024-04-09 Edited on 2025-11-01 Views:

用微软的 xiaoxiao natural 版本听书，当前来看已经很不错了，尤其是 edge 里可以直接使用。虽然来到这个时间节点，微软又推出了 azure 里几个新的模型，声音仿真效果真的变态好，不过全面铺开使用我看还得一段时间至少一年或者更长时间，尤其是开放到 edge 选项里估计更久。好在当前的已经很不错了。

本来用的 edge 里的一个插件EpubPup - EPUB Reader，可以导入打开 epub，试了下还是不错的，就是不能自动翻页。
但是苦于只能支持桌面版，所以又开始找手机版的方案。

然后发现了 flow，这个是一个浏览器的 epub 阅读器，支持手机版，而且开源，可以自己部署，这个是我想要的。后续还能自己加一些功能，比如自动翻页，这个是我想要的。
试了下 android 里可以用，iphone 里不行，而且还要选择 edge 里的打开桌面网页版才能正确使用 edge 语音。

✅ windows edge
✅ android edge，打开桌面网页版
❌ iphone edge

因为刚好有个安卓机，这样的组合基本满足了当前的需求，差点去买一个 windows 平板，现在看来是不用了。
