macOS 14 后字体模糊解决方法

Posted on 2020-08-19 Edited on 2025-12-19 Views:

可能是14之后吧，发现升级后mbp反而屏幕显示的没那么锐利了，明明是一样的retina屏幕。当然我自己测试后发现，iMac和mbp表现不一致，mbp可能不用设置或者设置 light font smoothing 就可以了，以各位的眼睛为准。

方法

  1. 第一步，开启 Subpixel Antialiasing。打开终端，输入如下命令。

    defaults write -g CGFontRenderingFontSmoothingDisabled -bool NO

  2. 第二步，选择字体的粗度及锐利度。

    Light font smoothing (1):
$ defaults -currentHost write -globalDomain AppleFontSmoothing -int 1
Medium font smoothing (2):
$ defaults -currentHost write -globalDomain AppleFontSmoothing -int 2
Heavy font smoothing (3):
$ defaults -currentHost write -globalDomain AppleFontSmoothing -int 3
No font smoothing (0):
$ defaults -currentHost write -globalDomain AppleFontSmoothing -int 0

  3. 第三步，注销重新登录才能起作用。
