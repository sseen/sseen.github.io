在 windows10 上调节 Studio Display 的亮度

Posted on 2023-07-04 Edited on 2025-11-01 Views:

从去年年底到今年年初的一段时间，曾经尝试过在 windows10 上使用 Studio Display，但是由于无法调节亮度，所以放弃了。最近又想起来，于是又开始折腾了一下，终于找到了解决方案。

其实之前也已经尝试到最后一步了，由于不知道如何安装 msi 文件，总是报错，就像第二个参考链接一样。

一句话就是 misexec 命令的时候，最后的 bootstrapper.msi 文件的路径一定要是绝对路径，不能是相对路径。

更新

不需要 bootcamp 来调节的方法，使用这个库的工具 GitHub - Chris-Poland/SDBC，页面上就有安装说明，这里简述下。

  1. 需要额外下载一个 libusb-win32 library，下载后照着说明安装
  2. 然后下载 sdbc，打开软件后，然后配置
  3. 总共有三个滑动条，随便调节下你就会用了

步骤

  1. 使用 brigadier，可以在 github 上找到这个库。
  2. 运行 brigadier.exe -model MacPro7,1 ，下载驱动。
  3. 去到下载的文件夹，找到 bootcamp.msi 文件，右键属性，复制路径。
  4. 打开 cmd，输入 msiexec /i "bootcamp.msi的绝对路径"，然后回车。
