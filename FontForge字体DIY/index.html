FontForge字体DIY

Posted on 2022-01-24 Edited on 2024-07-25 Views: Disqus:

看到一款霞鹜文楷挺好看的，看能不能一起弄到 jetbrain mono 放在 code 环境里。

fontforge MacOS 打开方式

  • 因为新系统权限的限制，直接去这里双击解决 /Applications/FontForge.app/Contents/Resources/opt/local/bin/fontforge
  • 或者只是修改一种字体无需另外引入的话，直接选中字体文件，然后右键选择fontforge打开，也可以

字体DIY

描述 方法
改变字体大小 fontforge 全选 ➡️ element ➡️ transformations ➡️ transform ➡️ 下拉框选择 scale uniformly 然后百分比 110% 之类
改变字体大小之后基本上都要改间距，不然会很挤。可以使用 autowidth 解决，但是每个字符都要设置一遍，fontforge这个地方没有性能优化，每次导致cpu100%卡半天，耐心等吧。还好耐心可以，这里有个小技巧，选择的时候可以按照字符类型选择，比如cjk或者hanguo之类，这个还不错，不用拖动选择了。不知道 fontlab 性能如何，有时间试试看。 fontforge 全选 ➡️ Metrics ➡️ Auto Width
改变字体名称 fontforge ➡️ element ➡️ Font info ➡️ 修改这里的 font name, family name, name for humans等
合并字体 fontforge ➡️ element ➡️ Merge fonts
ligature 字体 合并字体时候这里走了弯路，可以用ligature字体合并其他字体，这样ligature就不会丢失
