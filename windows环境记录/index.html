windows 环境记录

Posted on 2023-10-04 Edited on 2025-01-10 Views: Disqus:

公司发的是 windows 电脑，之前一直用 mac，现在又要用 windows 了，记录下环境配置。
本来突发奇想是不是可以装个 linux 环境不也可以，因为实在受不了 powershell 环境，不过后来用了 wsl2，感觉还是可以的，然后 windows 办公套件的话 linux 又不方便感觉，所以还是算了。

powershell 配置

类似其他 shell 一样，有个启动配置文件 ~\OneDrive\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1，可以在这里配置一些启动命令，比如 oh-my-posh 的启动命令。

oh-my-posh

美化命令行提示符，可以在这里查看安装和配置。

常用命令

  • ii . 在当前目录打开资源管理器
  • #开头或者ctrl r搜索历史命令

常用软件

  • Vysor。可以在电脑上显示手机屏幕，还可以控制手机，用来截图还是方便的，iOS 和 Android 都支持。
  • Flameshot。截图工具，可以截图后编辑，这个也是超级好用，尤其是交互很棒，跨平台，mac、windows、linux 都支持。
  • Mactype。字体渲染工具，windows 的字体渲染实在是太难看了，这个工具可以让字体看起来更舒服，不过有点卡顿，不过还是可以接受的。
  • Snip.Do。一个类似 popclip 的工具，可以在选中文字后弹出菜单，然后选择操作，比如翻译、搜索等等，比不过 popclip 还是差点，不过也还可以。
  • Ditto。剪贴板工具，可以保存多个剪贴板，还可以搜索。
  • 7zip。就不说了鼎鼎大名。
  • Everything。文件搜索工具，可以快速搜索文件，比 windows 自带的快多了。很早之前就用过，怀旧意味。
  • BlueStacks。安卓模拟器，可以在电脑上运行安卓应用，性能是真的可以。
  • Geek uninstaller。卸载工具，可以卸载软件的同时清理残留文件，比 windows 自带的卸载程序好用多了。
  • Optimizer 一款好用的 Windows 高级配置工具，增强 Windows 上的隐私和安全性。建议在 Windows 新安装后使用，以实现最大的隐私和安全性效益。根据您的 Windows 版本，优化器还可以帮助您应用特定的系统调整。https://github.com/hellzerg/optimizer
  • sharex。截图工具，开源主要是能滚动截图。
  • pot 跨平台翻译 GitHub - pot-app/pot-desktop: 🌈一个跨平台的划词翻译和 OCR 软件 | A cross-platform software for text translation and recognize.
