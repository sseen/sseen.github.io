macOS新环境配置

Posted on 2020-10-02 Edited on 2025-11-01 Views:

也由于最近iMac重装的问题，导致了macOS新环境部署安装重复了多次，虽然之前也有换电脑会遇到这种情况，但是几年一次也没意识到这种重复搜索的动作，终于下决心记录一下。前提肯定是可以自由联网。

📟第一，配置命令行环境

由于想到用homebrew cask来安装app，那么terminal配色的先配置吧。

# 1 terminal theme
curl -O https://raw.githubusercontent.com/ahmetsulek/flat-terminal/master/Flat.terminal
# 2 oh-my-zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
# zshrc 配置
plugin (z git)
theme (agnoster)

📚第二，安装各种常用软件物料

homebrew cask安装各种 app

# font
brew tap homebrew/cask-fonts
brew cask install font-fira-code
brew cask install font-cascadia-code font-cascadia-code-pl

# app
brew cask install visual-studio-code sublime-text
brew cask install google-chrome firefox
brew cask install github dropbox jetbrains-toolbox
# java
brew tap adoptopenjdk/openjdk
brew cask install adoptopenjdk/openjdk/adoptopenjdk8
# node hexo
brew install node
npm install hexo-cli -g
# python 3
