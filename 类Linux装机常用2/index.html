fcitx 安装

用了最新的最火的个人发行版 mx linux 基于 debian 那就试试看，主要是支持 xfce，但是目前当下时间点2021-01-03 xfce 还有 hidpi 的问题，尤其是安装了第三方软件后，第三方软件不能缩放，例如 vlc 和 astrill。
然后就用了它的 kde 版本，整体感觉很好，知道为什么 mx 这么火了，它的一套 mx 工具集真是给力，用户不用再分头搜索整理了，然后系统各种配置了一些人性化的设置，而不是需要用户从头设置开始，开箱开心的用，就不折腾了，还配置了 conky，真是省心了。
目前还没有配置中文的环境，那么就安装一下吧。
fcitx 是 Free Chinese Input Toy for X 的简称。
