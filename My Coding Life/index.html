My Coding Life

Posted on 2025-03-17 Edited on 2025-12-19 Views:

这是一个长期记录的博文吧，记录下编程路上的一些事情。目前Ai出来后，编程的形态也发生了很大的变化，趁在这个事业完全结束之前的进化洪流中，记录下自己的一些感受。

能够持续使用 ai 的职员

来到了这个时代，代码已经廉价化了门槛低多了，各种开源的工具也多。vscode 完全取代当年付费软件的所有功能，有过之而无不及。
看着编辑器一顿生成的几个十几个文件，你调试下，然后跑起来，神奇啊。速度更快了，功能也更准确了。之前搜索，还要自己判断和试下，现在直接生成，大部分还是能用的。
现在看身边的人，都是 chatgpt 打开，然后一顿操作。

能够持续使用 google 的职员

转眼google已经不能用了，开始步入如何在任何情况下，流畅使用最先进搜索的阶段。问题很多都是搜索解决的，搜索引擎越好结果越准确，越能解决问题。期间加上 github 的开始使用，基本能解决大部分问题。还是纯手工解决，只是能部分拷贝了。
这个阶段，你看有人用百度有人用google，也能区别他们的水准基本。

不会百度的新人

刚参加工作的时候，主管让我用 c 写一个程序，我哼次哼次debug还用到了 gdb 没记错的话，各种不方便啊，最后一周还是两周忘了，给主管说完成了，主管轻轻的一句，才写完啊，然后随手打开了百度网页，说其他人早就搜完写完了。
也不是不会搜索，那个时候还能用google呢，只是学生气浓，觉得搜索是能力不足的行为，要自己纯手工敲打才对味。
这个时候的编辑器，支持列模式那是相当了不起的。

软盘学生

最后一代用过软盘的学生，这就不多说了吧，容易暴露哈。当时 c 语言大作业的时候，还没有自己的电脑，把代码写到纸上，然后上机课的时候敲进去，永远记得，运行直接通过时候，自己都不敢相信的场景。
